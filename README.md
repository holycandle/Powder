# 项目代号：[Powder]
**《思想道德与法治》社会实践微视频 - 基于 Ren'Py 的交互式叙事项目**

## 1. 项目简介
本项目通过 Ren'Py 引擎开发一款时长约 15-20 分钟的交互式视觉小说（AVG），旨在探讨大学生因压力误入吸毒陷阱的真实案例，反思法律意识与生命价值。

## 2. 技术架构与管理
- **开发引擎**: Ren'Py (基于 Python)
- **项目管理**: GitHub Projects (看板式管理)
- **开源策略**: 本仓库仅公开 `.rpy` 逻辑脚本。出于隐私与版权考虑，多媒体资产（图片、音频）不上传至远程仓库。

## 3. 团队分工 (6-12人团队)
- **程序/架构 (组长)**: 负责 Ren'Py 底层、逻辑分支集成及 GitHub 维护。
- **剧本/策划**: 协助润色文本，确保剧情逻辑严谨。
- **素材生产 (摄影/配音/后期)**: 产出符合规范的图片与音频素材。

### 命名规范 (重要)
提交素材前请确保符合以下格式，否则程序无法识别：
- **背景图**: `bg_场景名_时间.webp` (例: `bg_library_night.webp`)
- **立绘**: `角色名_姿态或表情.png` (例: `hero_anxious.png`)
- **音频**: `类型_序号.mp3` (例: `v01_001.mp3`)
